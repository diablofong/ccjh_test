# 屏縣公正國中 第三次段考線上練習

線上網址：https://diablofong.github.io/ccjh_test/

## 功能

- 6 份題卷（國文注音卷、國文注釋卷、地理、公民、歷史、健康教育）
- 每次作答題目順序隨機洗牌
- 完成後點「計算成績」即時顯示對錯與分數
- 注音題自動轉選擇題（A/B/C/D），其餘為填空輸入
- 答案輸入容錯（注音一/ㄧ、是非 O/○/X/×、選擇題全形半形大小寫）

## 題卷清單

| 科目 | 題數 | 滿分 |
|------|------|------|
| 國文 國字注音卷 | 100 | 100 |
| 國文 注釋卷 | 50 | 50 |
| 地理 重點整理填空 | 34 | 34 |
| 公民 重點整理填空 | 12 | 12 |
| 歷史 重點整理填空 | 15 | 15 |
| 健康教育 是非與選擇 | 50 | 100 |

## 技術架構

純靜態 HTML/CSS/JS，無框架無建置步驟，部署於 GitHub Pages。答案以 UTF-8 Base64 編碼儲存，防止直接從原始碼看到解答。
